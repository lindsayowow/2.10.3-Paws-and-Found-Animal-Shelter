# 2.10.3-Paws-and-Found-Animal-Shelter-
Spring Free Build - Episode 2  
Hello! Welcome back! We just got the following app requests via email today, and really need
them done ASAP. Pick one, build it, and share! Since you’ll be working with a partner, consider
how you may be able to divide and conquer or pair program it.  
Paws & Found Animal Shelter
Hi! I run a small animal shelter called Paws & Found. Right now we're tracking all our
animals on a whiteboard, which is a mess. I need something simple on a computer that
my staff can use.
Each animal has a name, a species, and an age — that's really all we need to keep
track of for now.
My staff should be able to open a page in their browser, fill out a little form, and add a
new animal to the system. Nothing fancy, just the fields and a submit button. Once
submitted, I want that data to save somewhere.
I also heard we might connect a mobile app down the road, so the person building it told
me to ask you to make the animal data available "through an API, as JSON." I don't
really know what that means but apparently you do!
Done when...
Staff can open a page and fill out a form to add a new animal
Submitting the form actually saves the animal
The full list of animals is available as JSON
A single animal can be looked up by ID as JSON
A new animal can be saved via the API directly
